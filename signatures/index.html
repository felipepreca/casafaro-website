— How to install

Two minutes per inbox.

```
The "Copy HTML" button copies the rendered signature directly to your clipboard with all formatting preserved. Paste it into your email signature settings.

Gmail

  1. Tap Copy HTML on your card above.
  2. In Gmail (desktop): Settings → See all settings → General → Signature.
  3. Create a new signature. Paste with ⌘V (Mac) or Ctrl+V (PC). The formatting will carry over.
  4. Set it as default for new emails and replies. Save.

Apple Mail (Mac)

  1. Open this page in Safari and tap Copy HTML.
  2. In Mail: Mail → Settings → Signatures.
  3. Create a new signature, paste in. Uncheck "Always match my default message font" — otherwise Apple Mail strips the formatting.
  4. Drag the signature onto your account in the left column.

Apple Mail (iPhone)

  1. iOS strips most formatting from custom signatures. The cleanest path: set up the signature on your Mac as above, then sync via iCloud or rely on the desktop version. Alternatively, paste the Plain Text version.

Outlook

  1. Tap Copy HTML.
  2. File → Options → Mail → Signatures.
  3. Paste in the editor. Outlook will preserve the table layout.

On dark mode

The signature carries its own paper-cream background, set via both the bgcolor attribute and inline CSS — the two methods every major email client respects. On Gmail, Apple Mail, Outlook, and Superhuman in dark mode, the card stays cream and the ink stays ink. Nothing inverts.

On rendering

The signature uses Georgia (serif) and Helvetica/Arial (sans) — both available on every machine and email client without webfont loading. The arch is inline SVG, so it scales crisply on retina screens. No images to "click to load."
```
